FCC First API Challenge: Timestamp Microservice

Objectives:
  • User can pass a string to verify whether or not that string is a valid unix timestamp or a date in natural language format. For example, December 31, 1970, or dec 31 70.
  • If the string is valid, the app returns both the Unix timestamp and the natural language format.
  • If the string is not valid, the app returns null for both the unix and natrural language properties.

Examples

Enter string into the URL address bar following the models below.

https://still-shore-32119.herokuapp.com//july 4 2011
https://still-shore-32119.herokuapp.com//1309737600

Expected output:

{ "unix": 1309737600, "natural": "July 4, 2011" }
